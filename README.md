🛒 Amazon Capstone Excel Project

A comprehensive data analysis of Amazon sales data using Microsoft Excel — covering data cleaning, exploratory analysis, and business insights through pivot tables and charts.

 📌 Project Overview

This capstone project was completed as part of a structured data analytics learning programme. It demonstrates end-to-end data analysis skills using Excel, from raw data handling to generating meaningful business insights from an Amazon sales dataset.


🎯 Objectives

- Clean and prepare raw Amazon sales data for analysis
- Perform exploratory data analysis (EDA) to identify patterns
- Build pivot tables to summarise sales by category, region, and time
- Visualise findings through Excel charts and dashboards
- Draw actionable business conclusions from the data


🛠️ Tools & Skills Used

| Tool            |  Purpose |
| Microsoft Excel | Full analysis environment |
| Pivot Tables    | Data summarisation & aggregation |
| Excel Charts    | Data visualisation |
| Formulas & Functions | VLOOKUP, IF, SUMIF, COUNTIF, TEXT |
| Data Cleaning   | Removing duplicates, handling blanks, formatting |


📂 File

| File | Description |
| `Amazon.capstone.excel.project.xlsx` | Full Excel workbook with raw data, cleaned data, analysis, and charts |


📈 Key Insights

- Identified top-selling product categories by revenue and order volume
- Analysed sales trends over time to detect peak periods
- Examined customer order patterns and average order values
- Summarised regional sales performance across markets


🚀 How to View

1. Download the `.xlsx` file from this repository
2. Open with Microsoft Excel (2016 or later recommended) or Google Sheets
3. Navigate through the worksheet tabs to explore each stage of the analysis
